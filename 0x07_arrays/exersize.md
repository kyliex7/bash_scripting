### 1. make the script that acts like a grocery bag.
#### example:
```
    $ ./grocery_bag
    what would you like to do?
    1. add an item to the grocery bag
    2. remove an item from the grocery bag
    3. list items in the grocery bag
    4. exit
    please enter your choice (1-4): 1
    enter the item to add: milk
    milk has been added to the grocery bag.

    what would you like to do?
    1. add an item to the grocery bag
    2. remove an item from the grocery bag
    3. list items in the grocery bag
    4. exit
    please enter your choice (1-4): 1
    enter the item to add: eggs
    eggs has been added to the grocery bag.

    what would you like to do?
    1. add an item to the grocery bag
    2. remove an item from the grocery bag
    3. list items in the grocery bag
    4. exit
    please enter your choice (1-4): 3
    items in your grocery bag:
    - milk
    - eggs

    what would you like to do?
    1. add an item to the grocery bag
    2. remove an item from the grocery bag
    3. list items in the grocery bag
    4. exit
    please enter your choice (1-4): 2
    enter the item to remove: milk
    milk has been removed from the grocery bag.

    what would you like to do?
    1. add an item to the grocery bag
    2. remove an item from the grocery bag
    3. list items in the grocery bag
    4. exit
    please enter your choice (1-4): 3
    items in your grocery bag:
    - eggs

```
